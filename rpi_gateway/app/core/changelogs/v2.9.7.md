## v2.9.7 (2026-03-24) - Humidity Priority and Control Cleanup
Priority: high

The control screens now focus on the fruiting room, while the humidifier system and exhaust handling work in a safer order.

### Improved
- Exhaust now takes priority when air quality needs to be corrected.
- Fresh-air intake remains active during venting to support circulation.

### Fixed
- Spawning and device fan controls are no longer shown in the web controls page.
- Spawning room automation stays disabled while those fan controls are unavailable.
- Humidification no longer starts at the same time as exhaust ventilation.

### Changed
- The web controls page now shows only the active fruiting room controls.
