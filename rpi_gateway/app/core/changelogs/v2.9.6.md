## v2.9.6 (2026-03-22) - Real-Time AI Insights and Auto Control Fixes
Priority: high

Added real-time AI decision logging and actuator control improvements so the web UI and gateway reflect live ML behavior more accurately.

### Added
- AI Insights now surfaces recent ML decisions, current sensor snapshots, and active alert counts.
- Automatic control mode state is now persisted through user preferences.
- Active alerts are mirrored to Firebase and MQTT when available for better notification delivery.

### Fixed
- Manual actuator commands are blocked while automatic control is enabled.
- The actuator state API now reports real auto/manual status per actuator.
- Spawning-room alerts now flow through the alert pipeline alongside fruiting-room alerts.
- Real-time ML decisions are now written into the system logs for dashboard history.
