## v2.0.0 (2026-02-09) - Complete IoT Gateway
Priority: high

Major milestone: Complete IoT Gateway implementation with full-featured web dashboard, cloud integration, and automation engine.

### New
- Version info API endpoint for frontend display
- Enhanced settings page with system information
- Device identity management system
- User preferences persistence
- Comprehensive error handling across all modules
- Multi-room actuator state management
- Real-time connection status indicators
- WiFi mode detection and QR code generation
- Sensor warmup period with countdown timer

### Improved
- Web routes refactored with proper separation of concerns
- Enhanced logging with contextual message tags
- Better WiFi permission handling for NetworkManager
- Improved serial communication reliability
- Optimized database query performance
- Consolidated configuration management

### Breaking
- Updated API response format for consistency across endpoints
- Changed configuration file structure
- Renamed environment variables (see .env.example for new names)

### Known Limitations
- MQTT client not fully integrated (fallback mode only)
- OTA firmware updates not yet implemented
- Advanced failure recovery mechanisms are minimal
