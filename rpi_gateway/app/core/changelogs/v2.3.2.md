## v2.3.2 (2026-02-11) - Settings UI Overhaul
Priority: medium

Complete redesign of the System Settings page for improved usability and modern aesthetics.

### Improved
- **Modern Sidebar-Tab Layout**: Organized settings into logical categories (General, WiFi, Cloud, Power, About).
- **Premium Design Aesthetics**: Implementation of glassmorphism, smooth transitions, and a curated dark-mode color palette.
- **Enhanced Modals**: Redesigned confirmation dialogs and modals with a premium feel and improved contrast.
- **Better Navigation**: Vertical navigation tabs allow for quicker access to specific settings without scrolling through a long list.
- **Mobile Responsiveness**: Improved layout for mobile devices with a collapsible navigation system.

### Changed
- Refactored settings page structure for better modularity.
- Updated global styles for settings-specific components.
