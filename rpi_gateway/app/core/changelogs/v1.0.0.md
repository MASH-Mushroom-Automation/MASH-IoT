## v1.0.0 (2026-02-03) - Initial Release
Priority: high

First stable release of the M.A.S.H. IoT Gateway with complete sensor monitoring, actuator control, and cloud integration.

### New
- Web dashboard with real-time sensor monitoring and condition indicators
- USB Serial communication with Arduino (JSON protocol)
- WiFi provisioning via hotspot mode for easy network setup
- Offline-first database with automatic cloud sync when connected
- Cloud integration for real-time data streaming
- Machine learning automation engine for environment control
- Smart humidifier controller that prevents moisture overshoot
- Display power management for energy saving
- Manual actuator control with override tracking
- Dual-sensor support for fruiting and spawning room monitoring
- Safety watchdog that disables actuators on communication loss

### Supported Hardware
- Raspberry Pi 3/4
- Arduino Uno R3 via USB serial
- SCD41 CO2/temperature/humidity sensors (dual chamber)
- 8-channel relay module for actuator control
