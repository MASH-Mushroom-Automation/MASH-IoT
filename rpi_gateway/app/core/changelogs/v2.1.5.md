## v2.1.5 (2026-02-10) - Firebase Sync Improvements
Priority: medium

Fixed cloud sync toggle to work reliably across all connected dashboards and added diagnostic tools.

### New
- Server-side sync preference that applies to all connected dashboards
- Sync status endpoint for checking current sync state
- Sync toggle endpoint for enabling/disabling cloud sync from any client
- Firebase diagnostics endpoint for troubleshooting sync issues
- Detailed upload logging with success and error tracking

### Fixed
- Cloud sync toggle now works across multiple browser windows
- Removed dependency on browser local storage for sync state
- Fixed timing errors in dashboard cloud initialization

### Improved
- Cloud upload now checks user preference before attempting sync
- Better error handling with detailed logging
- Clearer status messages for cloud sync operations
