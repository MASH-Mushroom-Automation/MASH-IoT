### v2.3.1 (2026-02-11)
**Release Name:** OTA & Asset System
**Priority:** High

#### New
- **OTA Update System:** Implemented full Over-The-Air update mechanism using GitHub Releases.
  - Automatic update checks via `/api/update/check`.
  - One-click installation via `/api/update/install`.
  - **A/B Rollback:** Automatically reverts to the previous version if the new update fails health checks.
- **Asset Management:** Moved project assets to `rpi_gateway/app/web/static/assets` to ensuring they are included in OTA updates.

#### Improved
- **Web Interface:** Updated `base.html` and `routes.py` to serve assets correctly from the static directory.
- **Security:** Added GitHub Personal Access Token (PAT) support for authenticated private repository access.

#### Fixed
- **Changelogs:** Fixed changelog display to correctly load version-specific files.
