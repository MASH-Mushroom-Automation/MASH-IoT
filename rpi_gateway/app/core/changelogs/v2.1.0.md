## v2.1.0 (2026-02-10) - Firebase Realtime Database Integration
Priority: medium

Added real-time sensor data updates via cloud database for instant dashboard synchronization across all devices.

### New
- Real-time sensor value updates on dashboard without page refresh
- Cloud database integration for instant multi-device synchronization
- Device identification for cloud data routing
- Unique sensor card identifiers for live data updates
- Console logging for cloud sync debugging

### Improved
- Dashboard responsiveness with live data streaming
- Instant feedback when sensor values change

### Known Limitations
- Cloud API key must be manually configured
- Requires cloud project with Realtime Database enabled
