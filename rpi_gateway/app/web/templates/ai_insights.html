AI Features

  • Real-time Anomaly Detection: Continuously monitors sensor readings to detect unusual patterns and sensor faults
  • Fruiting Priority Logic: Exhaust ventilation activates before humidification when CO2 is too high
  • Spawn Run Guidance: Spawning remains sensor-only so high CO2 can support mycelial growth
  • Learning from Data: Models improve over time as they observe more cultivation cycles

Configuration

ML features can be enabled/disabled in config/config.yaml: 

system:
  ml_enabled: true  # Set to false to disable ML features
fruiting_room:
  co2_max: 1000
spawning_room:
  co2_max: 20000

Models are automatically trained and updated based on historical sensor data from both fruiting and spawning conditions.
